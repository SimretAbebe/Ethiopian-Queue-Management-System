## Queue Management System

A full-stack web application designed to manage service queues for professional institutions in a simple, efficient, and user-friendly way.

## Purpose

To reduce waiting time, improve service flow, and digitize queue handling for public institutions.

## Tech Stack

 Backend: Django, Django REST Framework

Database: SQLite (development), PostgreSQL (production)

Frontend: Modern JavaScript framework (React and typescript)

Styling: Clean UI with black text ,animations

Authentication: Token/JWT based authentication

## Features

User authentication and role management

Queue creation and management

Department-based services (using customizable institution names)

API-driven backend for frontend integration
